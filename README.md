

#  Fagner Felipe Correa — Portfólio

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido para apresentar projetos, habilidades e experiências na área de tecnologia, com foco em **Ciência de Dados, automação e governança de dados**.

---

##  Sobre o Projeto

O portfólio foi criado com o objetivo de consolidar minha presença profissional e demonstrar, de forma prática, minhas competências técnicas e analíticas.

A aplicação apresenta um design moderno, interativo e responsivo, priorizando **experiência do usuário (UX)** e **performance**.

---

##  Conteúdo

Neste portfólio, você encontrará:

*  Projetos de análise e qualidade de dados (QA)
*  Automação de processos e workflows (HubSpot)
*  Documentação técnica
*  Projetos de desenvolvimento frontend
*  Projeto educacional de cibersegurança em Python

---

##  Tecnologias Utilizadas

* **HTML5**
* **CSS3** (Design System com variáveis)
* **JavaScript (Vanilla)**
* **Google Fonts**

---

##  Funcionalidades

* Cursor personalizado interativo
* Animações com **Intersection Observer**
* Efeito **reveal on scroll**
* Navegação com scroll suave
* Destaque dinâmico no menu ativo
* Efeito de parallax no background
* Layout responsivo para diferentes dispositivos

```bash
portfolio/

```

##  Como Executar

Clone o repositório:

```bash
git clone https://github.com/Fagner14/seu-repositorio.git
```

Acesse a pasta do projeto:

```bash
cd seu-repositorio
```

Abra o arquivo no navegador:

```bash
index.html
```

> Nenhuma dependência externa é necessária.
